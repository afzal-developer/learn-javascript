Ch 1 : Variables

Variables are the terms that are vary able;
Name="Afzal";
Age="19";
Qualification="Matric";

  • Variable names are case sensitive
  • Only letter digit underscore and $ is allowed in variable name
  • Only a letter underscore or $ should be a first character. Digit can't be first character of a variable name
  • Reserved words cannot be variable names

Reserved Words

Reserved words are those words/terms in javascript which has specific meanings, proper names not common names, like console is a proper name but Console is not a proper name

Let - Const - Var

We will let variable or const variable or var variable
